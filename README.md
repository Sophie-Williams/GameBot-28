Best used with QtCreator.

Requirements
============

* Qt 5.4.1 (tested)
* QtQuick 2.4 (tested)
* QtGraphicalEffects 1.0
* OpenCV 2.4.9
* devel files for building (including qmake)

Building from command line
==========================

```
# mkdir levelme_build
# cd levelme_build
# qmake-qt5 ../GameBot/levelme_coc.pro
# make
# ./levelme_coc
```
